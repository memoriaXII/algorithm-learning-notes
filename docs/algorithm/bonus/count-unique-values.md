```
Input: nums = [1,1,1,1,1,2]
Output: 2

```

### Example 2

```
Input: nums = [1,2,3,4,4,4,7,7,12,12,13]
Output: 7
```

### Example 3

```
Input: nums = []
Output: 0
```

Output: 4
